    Theorem 7.37. Singular Value Decomposition.

    Let \(A\) be an \(m\times n\) matrix with rank \(r\text{.}\) Then \(A\) can be written as
    🔗
    \begin{equation*} A = U\Sigma V^T = \begin{bmatrix} \vec u_1 \amp \cdots \amp \vec u_m \end{bmatrix} \begin{bmatrix} \sigma_1 \amp 0 \amp 0 \amp \cdots \amp 0 \\ 0 \amp \sigma_2 \amp 0 \amp \cdots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots \amp \vdots \\ 0 \amp 0 \amp \cdots \amp \sigma_r \amp 0 \\ 0 \amp 0 \amp \cdots \amp 0 \amp 0 \end{bmatrix}\begin{bmatrix} \vec v_1^T \\ \vdots \\ \vec v_n^T \end{bmatrix} \end{equation*}
    where
    🔗
    • \(U\) is an \(m\times m\) orthogonal matrix,
      🔗
    • \(V\) is an \(n\times n\) orthogonal matrix,
      🔗
    • \(\Sigma\) is an \(m\times n\) diagonal matrix with diagonal entries \(\sigma_1,\dots,\sigma_r\text{,}\) where \(\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r \gt 0\text{.}\)
      🔗
    🔗
    Note that Singular Value Decomposition applies to any matrix, not just symmetric or square matrices.
    🔗

    Example 7.38.

    Consider the the linear transformation \(T(\vec x) = A\vec x\text{,}\) where
    \begin{equation*} A = \begin{bmatrix} 3 \amp -1 \\ -1 \amp 3 \end{bmatrix}\text{.} \end{equation*}
    Note that \(A\) has an orthogonal diagonalization given by
    \begin{equation*} A = PDP^T = \frac{1}{\sqrt{2}} \begin{bmatrix}1 \amp 1\\ -1 \amp 1\end{bmatrix} \begin{bmatrix}4 \amp 0\\ 0 \amp 2\end{bmatrix}\dfrac{1}{\sqrt{2}}\begin{bmatrix} 1 \amp -1 \\ 1 \amp 1 \end{bmatrix}\text{.} \end{equation*}
    Also, note that the orthogonal diagonalization of \(A\) satisfies the properties of a Singular Value Decomposition. The parts below investigate how multiplication by \(A\) acts on \(\mathbb{R}^2\) geometrically.
    🔗
    (a)
    Can you describe geometrically what multiplication by \(A\) does to an arbitrary vector \(\vec x\text{?}\)
    🔗
    Solution.
    Let’s see if we can figure out what \(T\) does to some nice inputs.
    🔗
    First, let’s see what happens to \(\vec e_1 = \begin{bmatrix} 1 \\ 0 \end{bmatrix}\text{.}\) Note that
    \begin{equation*} T\left(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\right) = \begin{bmatrix} 3 \amp -1 \\ -1 \amp 3 \end{bmatrix}\begin{bmatrix} 1 \\ 0 \end{bmatrix} = \begin{bmatrix} 3 \\ -1 \end{bmatrix}\text{.} \end{equation*}
    Geometrically, we can describe the action of \(T\) on this vector by scaling the input by \(\sqrt{10}\) and rotating it clockwise by some angle \(\theta\text{.}\)
    🔗
    Next, let’s see what happens to \(\vec e_2 = \begin{bmatrix} 0 \\ 1 \end{bmatrix}\text{.}\) Note that
    \begin{equation*} T\left(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\right) = \begin{bmatrix} 3 \amp -1 \\ -1 \amp 3 \end{bmatrix}\begin{bmatrix} 0 \\ 1 \end{bmatrix} = \begin{bmatrix} -1 \\ 3 \end{bmatrix}\text{.} \end{equation*}
    Geometrically, we can describe the action of \(T\) on this vector by scaling the input by \(\sqrt{10}\) and rotating it counterclockwise by that same angle \(\theta\text{.}\)
    🔗
    Now, let’s see what happens to \(\vec e_1 + \vec e_2 = \begin{bmatrix} 1 \\ 1 \end{bmatrix}\text{.}\) Note that
    \begin{equation*} T\left(\begin{bmatrix} 1 \\ 1 \end{bmatrix}\right) = \begin{bmatrix} 3 \amp -1 \\ -1 \amp 3 \end{bmatrix}\begin{bmatrix} 1 \\ 1 \end{bmatrix} = \begin{bmatrix} 2 \\ 2 \end{bmatrix}\text{.} \end{equation*}
    Geometrically, we can describe the action of \(T\) on this vector by scaling the input by \(2\) and no rotation.
    🔗
    It is very hard to describe what \(T\) is doing in general that will match the descriptions we found for just these three inputs.
    🔗
    🔗
    🔗
    (b)
    Instead of determining what \(A\) does in one fell swoop, let us investigate what each factor of \(A =PDP^T\) does. First, determine what multiplication by \(P^T\) does to an arbitrary vector \(\vec x\text{?}\)
    🔗
    Solution.
    Note that
    \begin{equation*} P^T = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \amp -1 \\ 1 \amp 1 \end{bmatrix} = \begin{bmatrix} \frac{\sqrt{2}}{2} \amp -\frac{\sqrt{2}}{2} \\ \frac{\sqrt{2}}{2} \amp \frac{\sqrt{2}}{2} \end{bmatrix} = \begin{bmatrix} \cos\left(45^\circ\right) \amp -\sin\left(45^\circ\right) \\ \sin\left(45^\circ\right) \amp \cos\left(45^\circ\right) \end{bmatrix}\text{.} \end{equation*}
    Therefore, multiplication by \(P^T\) rotates vectors \(45^\circ\) counterclockwise.
    🔗
    🔗
    🔗
    (c)
    Next, determine what multiplication by \(D\) does to an arbitrary vector \(\vec x\text{?}\)
    🔗
    Solution.
    Note that
    \begin{equation*} D\vec x = \begin{bmatrix} 4 \amp 0 \\ 0 \amp 2 \end{bmatrix}\begin{bmatrix} x_1 \\ x_2 \end{bmatrix} = \begin{bmatrix} 4x_1 \\ 2x_2 \end{bmatrix}\text{.} \end{equation*}
    Thus, multiplication by \(D\) scales the vector in the \(x_1\)-direction by \(4\) and in the \(x_2\)-direction by \(2\text{.}\)
    🔗
    🔗
    🔗
    (d)
    Now, determine what multiplication by \(P\) does to an arbitrary vector \(\vec x\text{?}\)
    🔗
    Solution.
    Note that
    \begin{equation*} P = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \amp 1 \\ -1 \amp 1 \end{bmatrix} = \begin{bmatrix} \frac{\sqrt{2}}{2} \amp \frac{\sqrt{2}}{2} \\ -\frac{\sqrt{2}}{2} \amp \frac{\sqrt{2}}{2} \end{bmatrix} = \begin{bmatrix} \cos\left(-45^\circ\right) \amp -\sin\left(-45^\circ\right) \\ \sin\left(-45^\circ\right) \amp \cos\left(-45^\circ\right) \end{bmatrix}\text{.} \end{equation*}
    Therefore, multiplication by \(P\) rotates vectors \(45^\circ\) clockwise.
    🔗
    🔗
    🔗
    (e)
    Combining parts (b) - (d) above, give a geometric description of what multiplication by \(A\) does to an arbitrary vector \(\vec x\text{.}\)
    🔗
    Solution.
    Mutliplcation by \(A\) does the following:
    1. First, it rotates the vector \(45^\circ\) counterclockwise.
      🔗
      🔗
    2. Then, scales the resulting vector by \(4\) in the \(x_1\)-direction and by \(2\) in the \(x_2\)-direction.
      🔗
      🔗
    3. Finally, rotates the resulting vector \(45^\circ\) clockwise.
      🔗
      🔗
    🔗
    🔗
    🔗
    🔗
    Recall from Theorem 6.34 that multiplication by an orthogonal matrix does not change the angles between vectors nor the vector’s length. Therefore, if \(U\) is an orthogonal matrix, we have that the linear transformation \(T(\vec x) = U\vec x\) must simply be a rotation and/or reflection.
    🔗
    Note that multiplication by a \(m \times n\) diagonal matrix \(\Sigma\) changes the vector from being in \(\mathbb{R}^n\) to being in \(\mathbb{R}^m\text{,}\) and then scales it along the appropriate coordinate axes.
    🔗
    Therefore, in general, we can use the SVD of a matrix \(A\) to describe how the linear transformation \(T(\vec x) = A\vec x\) acts on the vectors in \(\mathbb{R}^n\text{.}\)
    🔗
     
    🔗

    Note 7.39. Algorithm for Finding the SVD of \(A\).

    How do we find the SVD of a matrix \(A\text{?}\)
    1. Start by forming the symmetric matrix \(A^TA\) and finding its eigenvalues \(\lambda_i\text{,}\) with the eigenvalues assigned in nonincreasing order. Set \(\sigma_i = \sqrt{\lambda_i}\text{.}\) These are the singular values of \(A\) and they are the entries on the diagonal of \(\Sigma\text{.}\) Fill any missing diagonal elements with 0.
      • Since \(A^TA\) is symmetric, we know from Theorem 7.18 that it has \(n\) real eigenvalues. A bonus fact is that for symmetric matrices of the form \(A^TA\text{,}\) the \(n\) eigenvalues must be nonnegative, so
        \begin{equation*} \lambda_1 \ge \lambda_2 \ge \cdots \ge \lambda_n \ge 0. \end{equation*}
        🔗
        🔗
      🔗
      🔗
    2. Next, find \(n\) normalized and orthogonal eigenvectors of \(A^TA\) and call them \(\vec v_i\text{,}\) with \(\vec v_i\) corresponding to eigenvalue \(\lambda_i\text{.}\) These vectors form the rows of \(V^T\text{.}\)
      • Make sure to find enough orthonormal eigenvectors for each repeated eigenvalue, which we know we can do by Theorem 7.18.
        🔗
        🔗
      • Alternatively, we can think of the \(\vec v_i\) as the columns of \(V\text{,}\) but since SVD uses \(V^T\text{,}\) it is usually easier to think with respect to \(V^T\text{.}\)
        🔗
        🔗
      🔗
      🔗
    3. Use the formula
      \begin{equation*} \vec u_i = \dfrac{1}{\sigma_i}A\vec v_i \end{equation*}
      to find the columns of \(U\text{.}\) If the rank of \(A\) is less than \(m\text{,}\) then this formula will not produce enough columns to fill \(U\text{.}\) We must find the remaining columns by extending the set \(\left\{\vec u_1, \dots, \vec u_r\right\}\) to an orthonormal basis of \(\mathbb{R}^m\text{.}\)
      • The calculated vectors \(\vec u_i\) will be normalized and orthogonal to one another as long as the original \(\vec v_i\) were.
        🔗
        🔗
      • If the rank of \(A\) is less than \(n\text{,}\) then some of the eigenvalues found in (1) will be 0, which results in singular values of 0. These singular values can not be used in the formula, since that would result in dividing by 0.
        🔗
        🔗
      🔗
      🔗
    🔗
    🔗

    Example 7.40.

    Construct the SVD of
    🔗
    \begin{equation*} A = \begin{bmatrix}3 \amp 8 \\ 0 \amp 3 \end{bmatrix}. \end{equation*}
    Solution.
    First, we calculate the symmetric matrix \(A^TA\text{.}\) Note that
    \begin{equation*} A^TA = \begin{bmatrix} 3 \amp 0 \\ 8 \amp 3 \end{bmatrix}\begin{bmatrix} 3 \amp 8 \\ 0 \amp 3 \end{bmatrix} = \begin{bmatrix} 9 \amp 24 \\ 24 \amp 73\end{bmatrix}\text{.} \end{equation*}
    🔗
    Next, we find the eigenvalues of \(A^TA\text{.}\) Note that
    \begin{equation*} \det(A^TA - \lambda I) = (9-\lambda)(73-\lambda) - 24^2 = (\lambda - 81)(\lambda - 1)\text{.} \end{equation*}
    Therefore, the eigenvalues of \(A^TA\) are \(\lambda_1 = 81\) and \(\lambda_2 = 1\text{.}\)
    🔗
    Next, we find the singular values of \(A\) which are the square roots of the eigenvalues of \(A^TA\text{.}\) Therefore, we have that
    \begin{equation*} \sigma_1 = \sqrt{\lambda_1} = 9 \end{equation*}
    and
    \begin{equation*} \sigma_2 = \sqrt{\lambda_2} = 1\text{.} \end{equation*}
    It follows that
    \begin{equation*} \Sigma = \begin{bmatrix} \sigma_1 \amp 0 \\ 0 \amp \sigma_2 \end{bmatrix} = \begin{bmatrix} 9 \amp 0 \\ 0 \amp 1 \end{bmatrix}\text{.} \end{equation*}
    🔗
    Next, we find corresponding eigenvectors for each eigenvalue of \(A^TA\text{.}\) Note that
    \begin{equation*} A^TA - 81I = \begin{bmatrix} -72 \amp 24 \\ 24 \amp -8 \end{bmatrix} \sim \begin{bmatrix} 3 \amp -1 \\ 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    Thus,
    \begin{equation*} \vec v_1 = \dfrac{1}{\sqrt{10}}\begin{bmatrix} 1 \\ 3 \end{bmatrix} \end{equation*}
    is a unit eigenvector of \(A^TA\) corresponding to \(\lambda_1 = 81\text{.}\) Also, note that
    \begin{equation*} A^TA - I = \begin{bmatrix} 8 \amp 24 \\ 24 \amp 72 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 3 \\ 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    It follows that
    \begin{equation*} \vec v_2 = \dfrac{1}{\sqrt{10}}\begin{bmatrix} 3 \\ -1 \end{bmatrix} \end{equation*}
    is a unit eigenvector of \(A^TA\) corresponding to \(\lambda_2 = 1\text{.}\) (Note that we could have found \(\vec v_2\) in an alternative way by using Theorem 7.15.) These eigenvectors are the rows of \(V^T\text{.}\) Therefore, we have that
    \begin{equation*} V^T = \begin{bmatrix} \vec v_1^T \\ \vec v_2^T \end{bmatrix} = \begin{bmatrix} \frac{1}{\sqrt{10}} \amp \frac{3}{\sqrt{10}} \\ \frac{3}{\sqrt{10}} \amp \frac{-1}{\sqrt{10}}\end{bmatrix} = \frac{1}{\sqrt{10}}\begin{bmatrix} 1 \amp 3 \\ 3 \amp -1 \end{bmatrix}\text{.} \end{equation*}
    🔗
    Lastly, we find the columns of \(U\) by using the formula
    \begin{equation*} \vec u_i = \dfrac{1}{\sigma_i}A\vec v_i\text{.} \end{equation*}
    Note that
    \begin{equation*} \vec u_1 = \frac{1}{\sigma_1}A\vec v_1 = \frac{1}{9}\begin{bmatrix} 3 \amp 8 \\ 0 \amp 3 \end{bmatrix}\frac{1}{\sqrt{10}}\begin{bmatrix} 1 \\ 3 \end{bmatrix} = \frac{1}{9\sqrt{10}}\begin{bmatrix} 27 \\ 9 \end{bmatrix} = \frac{1}{\sqrt{10}}\begin{bmatrix} 3 \\ 1 \end{bmatrix} \end{equation*}
    and
    \begin{equation*} \vec u_2 = \frac{1}{\sigma_2}A\vec v_2 = \frac{1}{1}\begin{bmatrix} 3 \amp 8 \\ 0 \amp 3 \end{bmatrix}\frac{1}{\sqrt{10}}\begin{bmatrix} 3 \\ -1 \end{bmatrix} = \frac{1}{\sqrt{10}}\begin{bmatrix} 1 \\ -3 \end{bmatrix}\text{.} \end{equation*}
    Therefore, we have that
    \begin{equation*} U = \begin{bmatrix} \vec u_1 \amp \vec u_2 \end{bmatrix} = \begin{bmatrix} \frac{3}{\sqrt{10}} \amp \frac{1}{\sqrt{10}} \\ \frac{1}{\sqrt{10}} \amp -\frac{3}{\sqrt{10}}\end{bmatrix} = \frac{1}{\sqrt{10}}\begin{bmatrix} 3 \amp 1 \\ 1 \amp -3 \end{bmatrix}\text{.} \end{equation*}
    🔗
    We can check our Singular Value Decomposition by verifying that the matrices \(U, \Sigma,\) and \(V^T\) have the appropriate properties and satisfy \(A = U\Sigma V^T\text{.}\) Note that
    \begin{align*} U\Sigma V^T \amp = \frac{1}{\sqrt{10}}\begin{bmatrix} 3 \amp 1 \\ 1 \amp -3 \end{bmatrix}\begin{bmatrix} 9 \amp 0 \\ 0 \amp 1 \end{bmatrix}\frac{1}{\sqrt{10}}\begin{bmatrix} 1 \amp 3 \\ 3 \amp -1 \end{bmatrix}\\ \amp = \frac{1}{10}\begin{bmatrix} 3 \amp 1 \\ 1 \amp -3 \end{bmatrix}\begin{bmatrix} 9 \amp 27 \\ 3 \amp -1 \end{bmatrix}\\ \amp = \frac{1}{10}\begin{bmatrix}30 \amp 80 \\ 0 \amp 30 \end{bmatrix}\\ \amp = \begin{bmatrix} 3 \amp 8 \\ 0 \amp 3 \end{bmatrix} = A \end{align*}
    🔗
    🔗
    🔗

    Example 7.41.

    Construct the SVD of
    🔗
    \begin{equation*} A=\begin{bmatrix}1 \amp -1\\ -2 \amp 2\\ 2 \amp -2\end{bmatrix}. \end{equation*}
    Solution.
    First, note that
    \begin{equation*} A^TA = \begin{bmatrix} 1 \amp -2 \amp 2 \\ -1 \amp 2 \amp -2 \end{bmatrix}\begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \\ 2 \amp -2 \end{bmatrix} = \begin{bmatrix} 9 \amp -9 \\ -9 \amp 9 \end{bmatrix}\text{.} \end{equation*}
    We have that
    \begin{equation*} \det(A^TA - \lambda I) = (9-\lambda)^2 - 81 = \lambda^2 - 18\lambda = \lambda(\lambda - 18)\text{.} \end{equation*}
    So, the eigenvalues of \(A^TA\) are \(\lambda_1 = 18\) and \(\lambda_2 = 0\text{.}\) It follows that the singular values of \(A\) are \(\sigma_1 = \sqrt{18} = 3\sqrt{2}\) and \(\sigma_2 = 0\text{.}\) Recall that \(\Sigma\) has the same dimensions as \(A\text{.}\) Thus,
    \begin{equation*} \Sigma = \begin{bmatrix} 3\sqrt{2} \amp 0 \\ 0 \amp 0 \\ 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    Since there is only one nonzero singular value, we have that the rank of \(A\) is 1.
    🔗
    Note that
    \begin{equation*} A^TA - 18 I = \begin{bmatrix} -9 \amp -9 \\ -9 \amp -9 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 1 \\ 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    Therefore,
    \begin{equation*} \vec v_1 = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ -1 \end{bmatrix} \end{equation*}
    is a unit eigenvector of \(A^TA\) corresponding to an eigenvalue of \(\lambda_1 = 18\text{.}\) Also, note that
    \begin{equation*} A^TA - 0I = \begin{bmatrix} 9 \amp -9 \\-9 \amp 9 \end{bmatrix} \sim \begin{bmatrix} 1 \amp -1 \\ 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    Therefore,
    \begin{equation*} \vec v_2 = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 1 \end{bmatrix} \end{equation*}
    is a unit eigenvector of \(A^TA\) corresponding to an eigenvalue of \(\lambda_2 = 0\text{.}\) it follows that
    \begin{equation*} V^T = \dfrac{1}{\sqrt{2}}\begin{bmatrix} 1 \amp -1 \\ 1 \amp 1 \end{bmatrix}\text{.} \end{equation*}
    🔗
    We can find \(\vec u_1\) using the formula since \(\sigma_1 \neq 0\text{.}\) Note that
    \begin{equation*} \vec u_1 = \frac{1}{\sigma_1}A\vec v_1 = \frac{1}{3\sqrt{2}}\begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \\ 2 \amp -2 \end{bmatrix}\frac{1}{\sqrt{2}}\begin{bmatrix}1 \\ -1 \end{bmatrix} = \frac{1}{6}\begin{bmatrix} 2 \\ -4 \\ 4 \end{bmatrix} = \frac{1}{3}\begin{bmatrix} 1 \\ -2 \\ 2 \end{bmatrix}\text{.} \end{equation*}
    Note that we cannot use the formula to find \(\vec u_2\) since that would require us to divide by \(\sigma_2 = 0\text{.}\) However, we can find \(\vec u_2\) by using the properties of \(U\text{.}\) Recall \(U\) is a \(3 \times 3\) orthogonal matrix. So, we actually need to find two more columns of \(U\text{.}\) Thankfully, we can choose any two unit vectors as long as they make \(U\) an orthogonal matrix.
    🔗
    One set of vectors could be
    \begin{equation*} \vec u_2 = \frac{1}{\sqrt{2}}\begin{bmatrix} 0 \\ 1 \\ 1 \end{bmatrix} \quad \text{and} \quad \vec u_3 = \frac{1}{3\sqrt{2}}\begin{bmatrix} 4 \\ 1 \\ -1 \end{bmatrix}\text{.} \end{equation*}
    Let’s check that the matrix with these columns, call it \(U_1\text{,}\) works. Note that
    \begin{align*} U_1\Sigma V^T\amp = \begin{bmatrix} \frac{1}{3} \amp 0 \amp \frac{4}{3\sqrt{2}} \\ -\frac{2}{3} \amp \frac{1}{\sqrt{2}} \amp \frac{1}{3\sqrt{2}} \\ \frac{2}{3} \amp \frac{1}{\sqrt{2}} \amp -\frac{1}{3\sqrt{2}} \end{bmatrix}\begin{bmatrix} 3\sqrt{2} \amp 0 \\ 0 \amp 0 \\ 0 \amp 0 \end{bmatrix}\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \amp -1 \\ 1 \amp 1 \end{bmatrix}\\ \amp = \begin{bmatrix} \frac{1}{3} \amp 0 \amp \frac{4}{3\sqrt{2}} \\ -\frac{2}{3} \amp \frac{1}{\sqrt{2}} \amp \frac{1}{3\sqrt{2}} \\ \frac{2}{3} \amp \frac{1}{\sqrt{2}} \amp -\frac{1}{3\sqrt{2}} \end{bmatrix}\begin{bmatrix} 3 \amp -3 \\ 0 \amp 0 \\ 0 \amp 0 \end{bmatrix}\\ \amp = \begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \\ 2 \amp -2 \end{bmatrix} = A \end{align*}
    🔗
    The previous choice of \(\vec u_2\) and \(\vec u_3\) are not the only vectors that could fill the columns of \(U\text{.}\) Suppose instead we chose
    \begin{equation*} \vec u_2' = \frac{1}{3}\begin{bmatrix} 2 \\ -1 \\ -2 \end{bmatrix} \quad \text{and} \quad \vec u_3' = \frac{1}{3}\begin{bmatrix} 2 \\ 2 \\ 1 \end{bmatrix} \end{equation*}
    as the columns of \(U\text{,}\) call the matrix \(U_2\text{.}\) Then, we also have that \(A = U_2\Sigma V^T\) is a valid Singular Value Decomposition of \(A\text{.}\) There are many valid factorizations when there is more than one missing column of \(U\text{.}\)

    The SVD has been applied to many modern applications in CS, engineering, and mathematics. Some applications are:
    • Constructing bases for the four fundamental subspaces
      🔗
    • Forming approximations of a matrix
      🔗
    • Estimating the rank and the condition number of a matrix
      🔗
    • Computing the pseudoinverse of a matrix
      🔗
    • Linear least-squares problems
      🔗
    • Non-linear least-squares problems
      🔗
    • Image compression
      🔗
    • Machine learning and data mining
      🔗
    • Facial Recognition
      🔗
    • Principal component analysis
      🔗
    🔗
    Students are expected to be familiar with the first two items. We will explore these two items below.

    Theorem 7.42.

    Let \(A\) be an \(m \times n\) matrix with rank \(r\) and with SVD \(A = U\Sigma V^T\text{,}\) where
    \begin{equation*} U = \begin{bmatrix} \vec u_1 \amp \cdots \amp \vec u_m \end{bmatrix}, \quad \Sigma = \begin{bmatrix} \sigma_1 \amp 0 \amp 0 \amp \cdots \amp 0 \\ 0 \amp \sigma_2 \amp 0 \amp \cdots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots \amp \vdots \\ 0 \amp 0 \amp \cdots \amp \sigma_r \amp 0 \\ 0 \amp 0 \amp \cdots \amp 0 \amp 0 \end{bmatrix}, \quad V^T = \begin{bmatrix} \vec v_1^T \\ \vdots \\ \vec v_n^T \end{bmatrix} \end{equation*}
    Then,
    1. \(A\vec v_i = \sigma_i \vec u_i\) for \(i = 1, \cdots, r\text{.}\)
      🔗
      🔗
    2. \(\left\{\vec v_1, \dots, \vec v_r\right\}\) is an orthonormal basis for \(\Row(A)\text{.}\)
      🔗
      🔗
    3. \(\left\{\vec u_1, \dots, \vec u_r\right\}\) is an orthonormal basis for \(\Col(A)\text{.}\)
      🔗
      🔗
    4. \(\left\{\vec v_{r+1}, \dots, \vec v_n\right\}\) is an orthonormal basis for \(\Nul(A)\text{.}\)
      🔗
      🔗
    5. \(\left\{\vec u_{r+1}, \dots, \vec u_m\right\}\) is an orthonormal basis for \(\Nul(A^T)\text{.}\)

    Example 7.43.

    Let \(A = U\Sigma V^T\) for
    \begin{equation*} U = \begin{bmatrix} 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp -1 \\ 1 \amp 0 \amp 0 \amp 0 \end{bmatrix}, \quad \Sigma = \begin{bmatrix} 4 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 3 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp \sqrt{5} \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}, \quad V^T = \begin{bmatrix} 0 \amp 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \amp 0 \\ \sqrt{0.2} \amp 0 \amp 0 \amp 0 \amp \sqrt{0.8} \\ 0 \amp 0 \amp 0 \amp 1 \amp 0 \\ -\sqrt{0.8} \amp 0 \amp 0 \amp 0 \amp \sqrt{0.2} \end{bmatrix}\text{.} \end{equation*}
    Find an orthonormal basis for each of the four subspaces.
    🔗
    Solution.
    Since \(\Sigma\) has three nonzero elements on its diagonal, we know that \(\rank(A) = 3\text{.}\) Therefore, the first 3 columns of \(U\) form an orthonormal basis for \(\Col(A)\) and the remaining column of \(U\) forms an orthonormal basis for \(\Nul(A)^T\text{.}\) Similarly, the first 3 rows of \(V^T\) form an orthonormal basis for \(\Row(A)\) and the remaining rows of \(V^T\) form an orthonormal basis for \(\Nul(A)\text{.}\) Therefore, we have that
    \begin{align*} \mathcal{B}_1 \amp = \left\{\begin{bmatrix} 0 \\ 0 \\ 0 \\ 1 \end{bmatrix}, \begin{bmatrix} 0 \\ 1 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}\right\} \text{ is an orthonormal basis for $\Col(A)$.}\\ \mathcal{B}_2 \amp = \left\{\begin{bmatrix} 0 \\ 0 \\ -1\\ 0 \end{bmatrix}\right\} \text{ is an orthonormal basis for $\Nul(A^T)$.}\\ \mathcal{B}_3 \amp = \left\{\begin{bmatrix} 0 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 0 \\ 1 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} \sqrt{0.2} \\ 0 \\ 0 \\ 0 \\ \sqrt{0.8} \end{bmatrix}\right\} \text{ is an orthonormal basis for $\Row(A)$.}\\ \mathcal{B}_4 \amp = \left\{\begin{bmatrix} 0 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix}, \begin{bmatrix} -\sqrt{0.8} \\ 0 \\ 0 \\ 0 \\ \sqrt{0.2} \end{bmatrix}\right\} \text{ is an orthonormal basis for $\Nul(A)$.} \end{align*}

    Theorem 7.44.

    Let \(A = U\Sigma V^T\) be the SVD of a rank \(r\) matrix, with the columns of \(U\) given by \(\vec u_i\text{,}\) the entries on the diagonal of \(\Sigma\) given by \(\sigma_i\text{,}\) and the rows of \(V^T\) given by \(\vec v_i\text{.}\) Then, the spectral decomposition of \(A\) is given by
    \begin{equation*} A=\sum_{i=1}^r \sigma_i\,\vec u_i\vec v_i^T. \end{equation*}

    Example 7.45.

    Find the spectral decomposition of \(A = \begin{bmatrix}3 \amp 8 \\ 0 \amp 3 \end{bmatrix}\text{.}\) Use the decomposition to identify the best rank 1 approximation of \(A\text{.}\)
    🔗
    Solution.
    Using the information from the solution to Example 7.40,we have that
    \begin{align*} \begin{bmatrix} 3 \amp 8 \\ 0 \amp 3 \end{bmatrix} \amp = \sigma_1\vec u_1 \vec v_1^T + \sigma_2 \vec u_2 \vec v_2^T\\ \amp = (9)\frac{1}{\sqrt{10}}\begin{bmatrix}3 \\ 1 \end{bmatrix} \frac{1}{\sqrt{10}}\begin{bmatrix} 1 \amp 3 \end{bmatrix} + (1)\frac{1}{\sqrt{10}}\begin{bmatrix} 1 \\ -3 \end{bmatrix}\frac{1}{\sqrt{10}}\begin{bmatrix} 3 \amp -1 \end{bmatrix} \\ \amp = \frac{9}{10}\begin{bmatrix} 3 \amp 9 \\1 \amp 3 \end{bmatrix} + \frac{1}{10}\begin{bmatrix} 3 \amp -1 \\ -9 \amp 3 \end{bmatrix}\\ \amp = \begin{bmatrix} 2.7 \amp 8.1 \\ 0.9 \amp 2.7 \end{bmatrix} + \begin{bmatrix} 0.3 \amp -0.1 \\ -0.9 \amp 0.3 \end{bmatrix}\text{.} \end{align*}
    Therefore, the best rank 1 matrix that approximates \(A\) is \(\begin{bmatrix} 2.7 \amp 8.1 \\ 0.9 \amp 2.7 \end{bmatrix}\text{.}\)

    We end this section, and this course, by investigating the connection between Spectral Decomposition and Image Compression. Consider an image in grayscale consisting of pixels in a \(880 \times 1100\) grid. Since each pixel is in grayscale, we can assign it an integer value between 0 and 255 depending on how gray it is, where 0 is black and 255 is white. We can form a \(880 \times 1100\) matrix where each entry is the grayscale value of the associated pixel. In general, the rank of this matrix could be quite high.
    🔗
    For example, suppose we have selected an image and created the associated matrix and it has rank \(776\text{.}\) One way to store this image in a slightly compressed manner is to store the \(776\) basis vectors for \(\Col(A)\text{,}\) and then store a list of linear combinations describing how to use these \(776\) vectors to recreate all \(880\) columns (the dimensions of an image are reversed, first horizontal, then vertical).

    Question 7.46.

    Could we compress the file size further? Suppose we were okay with a little error in our image. Perhaps just a hair of fuzziness that could be corrected open reopening the compressed file. How could we go about doing this?
    🔗
    🔗
    The idea is to find a good enough approximation of the image. We can use Spectral Decomposition to find the best rank \(j\) approximation for \(j = 1, \dots, r\text{.}\) We can save tons of storage by letting \(j\) be small, but then our image will be super blurry. We can have a very accurate image by letting \(j\) be large, but then the compression’s filesize is not much smaller than the original’s filesize. We need to find a \(j\) which balances these, a good approximation with small filesize.
