    Definition 1.41. Homogeneous and inhomogeneous systems.

    Linear systems of the form \(A\vec{x} = \vec{0}\) are homogeneous. Linear systems of the form \(A\vec{x} = \vec{b}\) (with \(\vec{b} \neq \vec{0}\)) are inhomogeneous.
    ðŸ”—
    ðŸ”—
    Because homogeneous systems always have the trivial solution \(\vec{x} = \vec{0}\text{,}\) the interesting question is whether they have any nontrivial solutions.
    ðŸ”—

    Example 1.42. Identify free variables and solution set.

    Identify the free variables and the solution set of the homogeneous system:
    ðŸ”—
    \begin{equation*} \begin{aligned} x_1 + 3x_2 + x_3 \amp= 0 \\ 2x_1 - x_2 - 5x_3 \amp= 0 \\ x_1 \phantom{+3x_2} - 2x_3 \amp= 0 \end{aligned} \end{equation*}
    Solution.
    Since this is a homogeneous system, we know it has the trivial solution \(\vec x = \vec 0\) as a solution. However, we want to find all solutions. So, letâ€™s solve the system by simplifying the augmented matrix.
    ðŸ”—
    Note: you do not have to form the augmented matrix and can instead row reduce \(A\) since this is a homogeneous system and row operations will not affected the augmented \(\vec 0\text{.}\)
    ðŸ”—
    \begin{equation*} \begin{bmatrix} 1 \amp 3 \amp 1 \amp \rvert \amp 0 \\ 2 \amp -1 \amp -5 \amp \rvert \amp 0 \\ 1 \amp 0 \amp -2 \amp \rvert \amp 0 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 3 \amp 1 \amp \rvert \amp 0 \\ 0 \amp -7 \amp -7 \amp \rvert \amp 0 \\ 0 \amp -3 \amp -3 \amp \rvert \amp 0 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 0 \amp -2 \amp \rvert \amp 0 \\ 0 \amp 1 \amp 1 \amp \rvert \amp 0 \\ 0 \amp 0 \amp 0 \amp \rvert \amp 0 \end{bmatrix} \end{equation*}
    ðŸ”—
    Since the third column does not contain a pivot, we have that \(x_3\) is a free variable, so we can set \(x_3 = t \in \mathbb{R}\text{.}\) Converting the second row of the RREF above into an equation and solving for \(x_2\) yields
    \begin{equation*} x_2 + x_3 = 0 \quad \implies \quad x_2 = -x_3 \quad \implies \quad x_2 = -t \end{equation*}
    Similarly, we can use the first row of the RREF and solve for \(x_1\) to get
    \begin{equation*} x_1 - 2x_3 = 0 \quad \implies \quad x_1 = 2x_3 \quad \implies \quad x_1 = 2t \end{equation*}
    ðŸ”—
    Therefore, the solutions to the system are
    ðŸ”—
    \begin{equation*} \vec x = \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \begin{bmatrix} 2t \\ -t \\ t \end{bmatrix} = t\begin{bmatrix} 2 \\ -1 \\ 1 \end{bmatrix}, t \in \mathbb{R} \end{equation*}
    ðŸ”—
    Writing the solution set in terms of the parameter \(t\) is what is known as parametric vector form. This form gives an explicit and convenient way of writing down the solution set. It also gives insight into what is happening geometrically. Note that the vectors which satisfy the system are precisely
    \begin{equation*} \vec x \in \textrm{Span}\left\{\begin{bmatrix} 2 \\ -1 \\ 1 \end{bmatrix}\right\}\text{.} \end{equation*}
    Since the solution set contains \(\vec 0\text{,}\) we see that the solution set is a line in \(\mathbb{R}^3\) which passes through the origin.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    In the previous example, we expressed the solution to the system using a vector equation. This is a parametric vector form of the solution.
    ðŸ”—
    In general, suppose the free variables for \(A\vec{x} = \vec{0}\) are \(x_k, \ldots, x_n\text{.}\) Then all solutions can be written in the form
    ðŸ”—
    \begin{equation*} \vec{x} = x_k \vec{v}_k + x_{k+1} \vec{v}_{k+1} + \cdots + x_n \vec{v}_n, \end{equation*}
    for some vectors \(\vec{v}_k, \ldots, \vec{v}_n\text{.}\) This representation is called the parametric vector form of the solution set.
    ðŸ”—

    Example 1.43. Parametric vector form and geometric interpretation.

    Write the parametric vector form of the solution and give a geometric interpretation.
    ðŸ”—
    \begin{equation*} \begin{aligned} x_1 + 3x_2 + x_3 \amp= 9 \\ 2x_1 - x_2 - 5x_3 \amp= 11 \\ x_1 \phantom{+3x_2} - 2x_3 \amp= 6 \end{aligned} \end{equation*}
    (Note: the left-hand side is the same as in the homogeneous system example.)
    ðŸ”—
    Solution.
    Again, we solve the system by simplifying the augmented matrix. This time we do need to keep track of the augmented column since it is not \(\vec 0\) and so row operations will affect the entries.
    ðŸ”—
    \(\begin{eqnarray*}\begin{bmatrix} 1 \amp 3 \amp 1 \amp \rvert \amp 9 \\ 2 \amp -1 \amp -5 \amp \rvert \amp 11 \\ 1 \amp 0 \amp -2 \amp \rvert \amp 6 \end{bmatrix} \amp \sim \amp \begin{bmatrix} 1 \amp 3 \amp 1 \amp \rvert \amp 9 \\ 0 \amp -7 \amp -7 \amp \rvert \amp -7 \\ 0 \amp -3 \amp -3 \amp \rvert \amp -3 \end{bmatrix} \\ \amp \sim \amp \begin{bmatrix} 1 \amp 0 \amp -2 \amp \rvert \amp 6 \\ 0 \amp 1 \amp 1 \amp \rvert \amp 1 \\ 0 \amp 0 \amp 0 \amp \rvert \amp 0 \end{bmatrix}\end{eqnarray*}\)
    ðŸ”—
    Thankfully, the third row had a zero in the augmented position, so our system is consistent and there is at least one solution. Notice that the entries to the left of the augmentation line are exactly the same as they were in ExampleÂ 1.42. Therefore, assigning free variables and solving for the pivot variables is almost exactly the same as in that example. The only difference, is now we have to keep track of the nonzero constants on the right side of the augmentation line. We have that \(x_3\) is a free variable, so we set \(x_3 = t \in \mathbb{R}\text{.}\) Solving for the pivot variables yields
    \begin{equation*} x_2 + x_3 = 1 \quad \implies \quad x_2 = 1 -x_3 \quad \implies \quad x_2 = 1 -t \end{equation*}
    and
    \begin{equation*} x_1 - 2x_3 = 6 \quad \implies \quad x_1 = 6 + 2x_3 \quad \implies \quad x_1 = 6 + 2t \end{equation*}
    ðŸ”—
    Therefore, the solutions to the system are
    ðŸ”—
    \begin{equation*} \vec x = \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \begin{bmatrix} 6 + 2t \\ 1 - t \\ t \end{bmatrix} = \begin{bmatrix} 6 \\ 1 \\ 0 \end{bmatrix} + t\begin{bmatrix} 2 \\ -1 \\ 1 \end{bmatrix}, t \in \mathbb{R} \end{equation*}
    ðŸ”—
    Again we have writen the solution set in parametric vector form. Note that the free part of the solution set, the part being scaled by \(t\text{,}\) is exactly the same as for the homogeneous system \(A\vec x = \vec 0\text{.}\) This is no coincidence. When you are working with a consistent inhomogeneous system \(A\vec x = \vec b\text{,}\) the solution set is exactly that of the homogenous system \(A\vec x = \vec 0\text{,}\) except translated by a vector which is a particular solution to \(A\vec x = \vec b\text{.}\) So, geometrically, this solution set is the same line as in ExampleÂ 1.42, except now the line passes through the point \((6,1,0)\) instead of the origin.
    ðŸ”—
    Note: if the inhomogeneous system \(A\vec x = \vec b\) is inconsistent, then there is still no solution no matter how many solutions there are to \(A\vec x = \vec 0\text{.}\)
