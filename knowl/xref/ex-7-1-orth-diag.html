Example 7.16.

Diagonalize \(A\) using an orthogonal matrix. The eigenvalues of \(A\) are given.
🔗
\begin{equation*} A=\begin{bmatrix} 0 \amp 0 \amp 1\\ 0 \amp 1 \amp 0\\ 1 \amp 0 \amp 0 \end{bmatrix}, \qquad \lambda=-1,\,1. \end{equation*}
Solution.
Let’s start with finding the \(1\)-eigenspace of \(A\text{.}\) Note that
\begin{equation*} A - I = \begin{bmatrix} -1 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 1 \amp 0 \amp -1 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 0 \amp -1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
Since \(A - I\) has two free variables, the \(1\)-eigenspace\((A)\) is two dimensional. If we were only trying to diagonalize \(A\text{,}\) then any basis of the \(1\)-eigenspace\((A)\) would be fine. But we want to diagonalize so that \(P\) is an orthogonal matrix. Therefore, we want to find an orthonormal basis of the \(1\)-eigenspace\((A)\text{.}\) Thus, we choose the basis vectors
\begin{equation*} \vec v_1 = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} \quad \text{and} \quad \vec v_2 = \dfrac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}\text{.} \end{equation*}
🔗
Now, let’s find a basis for the \(-1\)-eigenspace\((A)\text{.}\) Note that
\begin{equation*} A - (-1)I = \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 2 \amp 0 \\ 1 \amp 0 \amp 1 \end{bmatrix}\text{.} \end{equation*}
Since \(A - (-1)I\) has only a single free variable, any basis we choose will be an orthogonal basis, we will just make sure to use a normalized vector as a column of \(P\text{.}\) Let
\begin{equation*} \vec v_3 = \dfrac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ -1 \end{bmatrix}\text{.} \end{equation*}
Note that this vector is guaranteed to be orthogonal to \(\vec v_1\) and \(\vec v_2\) by Theorem 7.15.
🔗
It follows that a diagonalization of \(A\) is given by \(A = PDP^{-1}\text{,}\) where
\begin{equation*} D = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \end{bmatrix} \quad \text{and} \quad P = \begin{bmatrix}0 \amp \frac{1}{\sqrt{2}} \amp \frac{1}{\sqrt{2}} \\ 1 \amp 0 \amp 0 \\ 0 \amp \frac{1}{\sqrt{2}} \amp -\frac{1}{\sqrt{2}} \end{bmatrix}\text{.} \end{equation*}
Since \(P\) is orthogonal, we have that \(P^{-1} = P^T\text{,}\) so this diagonalization could be written \(A = PDP^T\text{.}\) We call this diagonalization using an orthogonal matrix an orthogonal diagonalization.
