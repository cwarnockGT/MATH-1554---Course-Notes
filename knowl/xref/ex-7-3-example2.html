Example 7.32.

Calculate the maximum and minimum values of
🔗
\begin{equation*} Q(\vec x)=x_2^2 + 2x_1x_3. \end{equation*}
where \(\vec x\in\mathbb{R}^3\) and \(\|\vec x\|=1\text{.}\) Identify the points where these values are obtained.
🔗
Solution.
Since \(Q(\vec x)\) has cross-product terms, we will make the change of variables \(\vec x = P\vec y\) to get an equivalent quadratic form without cross-product terms. Note that the unit vector constraint is preserved and stil imposed on \(\vec y\) since
\begin{equation*} 1 = \|\vec x\| = \|P\vec y\| = \|\vec y\| \end{equation*}
since \(P\) is orthogonal matrix and Theorem 6.34 tells us that multiplication preserves lengths.
🔗
We have that the symmetric matrix \(A\) so that \(Q_A(\vec x) = x_2^2 + 2x_1x_3\) is
\begin{equation*} A = \begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
In Example 7.16, we saw that the eigenvalues are \(\lambda_1 = 1\text{,}\) \(\lambda_2 = 1\text{,}\) and \(\lambda_3 = -1\) with accompanying unit eigenvectors
\begin{equation*} \vec v_1 = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}, \quad \vec v_2 = \dfrac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}, \quad \text{and} \quad \vec v_3 = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ -1 \end{bmatrix}\text{.} \end{equation*}
Therefore, setting \(P = \begin{bmatrix} \vec v_1 \amp \vec v_2 \amp \vec v_3 \end{bmatrix}\) and making the change of variables \(\vec x = P\vec y\text{,}\) we deduce
\begin{equation*} Q_A(\vec x) = Q_D(\vec y) = y_1^2 + y_2^2 - y_3^2\text{.} \end{equation*}
Thus, the maximum value of \(Q(\vec x) = x_2^2 + 2x_1x_3\) under the constraint that the input is a unit vector is \(1\text{,}\) and the minimum is \(-1\text{.}\) These maximum and minimum values occur at eigenvectors corresponding to the given eigenvalue. For example, note that
\begin{align*} Q_A(\vec v_1) \amp = (1)^2 + 2(0)(0) = 1\\ Q_A(\vec v_2) \amp = 0^2 + 2\left(\frac{1}{\sqrt{2}}\right)\left(\frac{1}{\sqrt{2}}\right) = 2\left(\frac{1}{2}\right) = 1\\ Q_A(\vec v_3) \amp = 0^2 + 2\left(\frac{1}{\sqrt{2}}\right)\left(-\frac{1}{\sqrt{2}}\right) = 2\left(-\frac{1}{2}\right) = -1 \end{align*}
