Theorem 6.33. Mapping Properties of Orthogonal Matrices.

If \(U\) is an orthogonal matrix, then:
🔗
  1. Preserves angles: \((U\vec x)\cdot(U\vec y)=\vec x\cdot\vec y\text{.}\)
    Solution.
    Using the defintion of dot product (Definition 6.1), we see that
    \begin{equation*} (U\vec x)\cdot (U\vec y) = (U\vec x)^T(U\vec y) = \vec x^T U^TU\vec y\text{.} \end{equation*}
    By Theorem 6.32, we see that \(U^TU = I_n\text{.}\) Therefore,
    \begin{equation*} (U\vec x)\cdot(U\vec y) = \vec x^TU^TU\vec y = \vec x^T\vec y = \vec x \cdot \vec y\text{.} \end{equation*}
    🔗
    🔗
    🔗
  2. Preserves length: \(\| U\vec x\| = \|\vec x\|\text{.}\)
    Solution.
    This is a direct consequence of using Definition 6.4 and the above result with \((U\vec x)\) and itself.
