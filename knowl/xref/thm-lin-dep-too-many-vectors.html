Theorem 1.47.

Suppose \(\vec v_1, \ldots, \vec v_n\) are vectors in \(\mathbb{R}^m\) with \(n > m\text{.}\) Then the set \(\{\vec v_1, \ldots, \vec v_n\}\) is linearly dependent.
Solution.
Note that the matrix \(A = \begin{bmatrix} \vec v_1 \amp \cdots \amp \vec v_k \end{bmatrix}\) is an \(m \times n\) matrix. Since \(n > m\text{,}\) this matrix has more columns than rows. Therefore, there must be a column that doesnâ€™t contain a pivot. Thus, the vectors must be linearly dependent.
