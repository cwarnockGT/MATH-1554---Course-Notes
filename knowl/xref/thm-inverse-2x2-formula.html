Theorem 2.17. Inverse Formula for a \(2\times 2\) Matrix.

The \(2\times 2\) matrix \(\begin{bmatrix} a \amp b\\ c \amp d \end{bmatrix}\) is invertible if and only if \(ad-bc\neq 0\text{.}\) In that case,
ðŸ”—
\begin{equation*} \begin{bmatrix} a \amp b\\ c \amp d \end{bmatrix}^{-1} = \frac{1}{ad-bc} \begin{bmatrix} d \amp -b\\ -c \amp a \end{bmatrix}. \end{equation*}

The idea is to start with a matrix \(A = \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\) for fixed \(a,b,c,d \in \mathbb{R}\) with \(ad - bc \neq 0\text{.}\) Now, let \(A^{-1} = \begin{bmatrix} x_1 \amp x_2 \\ x_3 \amp x_4 \end{bmatrix}\) for unknown \(x_1,x_2,x_3,x_4\in \mathbb{R}\text{.}\) Set \(AA^{-1} = I_2\) and determine what the unknowns \(x_1,x_2,x_3,x_4\) are in terms of the known \(a,b,c,d\text{.}\) You should end up solving a system of linear equations involving 4 equations in the 4 unknowns.

Solution.
Here is a proof of the above formula. (Before reading, you may want to try it yourself. Look at the hint above.)
ðŸ”—
Let \(A = \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\) for fixed \(a,b,c,d \in \mathbb{R}\) with \(ad - bc \neq 0\text{.}\) We are looking for a \(2 \times 2\) matrix \(A^{-1}\) so that \(AA^{-1} = I_2\text{,}\) if such a matrix exists. Since \(A^{-1}\) is a \(2 \times 2\) matrix, it has four unknown entries. Letâ€™s call them \(x_1, x_2, x_3, x_4 \in \mathbb{R}\text{,}\) so we have \(A^{-1} = \begin{bmatrix} x_1 \amp x_2 \\ x_3 \amp x_4 \end{bmatrix}\text{.}\) It follows that
\begin{equation*} AA^{-1} = \begin{bmatrix}a \amp b \\ c \amp d \end{bmatrix}\begin{bmatrix} x_1 \amp x_2 \\ x_3 \amp x_4 \end{bmatrix} = \begin{bmatrix} ax_1 + bx_3 \amp ax_2 + bx_4 \\ cx_1 + dx_3 \amp cx_2 + dx_4\end{bmatrix}\text{.} \end{equation*}
Since we want \(AA^{-1} = I_2\text{,}\) we can set the corresponding entries equal to arrive at the following system of linear equation:
\begin{equation*} \left\{\begin{array}{ccccccccc} ax_1 \amp \amp \amp + \amp bx_3 \amp \amp \amp = \amp 1 \\ \amp \amp ax_2 \amp \amp \amp + \amp bx_4 \amp = \amp 0 \\ cx_1 \amp \amp \amp + \amp dx_3 \amp \amp \amp = \amp 0 \\ \amp \amp cx_2 \amp \amp \amp + \amp dx_4 \amp = \amp 1 \end{array}\right . \end{equation*}
Note that we cannot have both \(a\) and \(c\) be zero, or else \(ad - bc = 0\text{.}\) Therefore, we can assume that at least one of them is nonzero, so letâ€™s suppose \(a \neq 0\text{.}\) Converting this into an augmented matrix and performing some row operations, we obtain
\begin{equation*} \begin{bmatrix}a \amp 0 \amp b \amp 0 \amp \rvert \amp 1 \\ 0 \amp a \amp 0 \amp b \amp \rvert \amp 0 \\ c \amp 0 \amp d \amp 0 \amp \rvert \amp 0 \\ 0 \amp c \amp 0 \amp d \amp \rvert \amp 1 \end{bmatrix} \underset{R_4 - \frac{c}{a}R_2 \rightarrow R_4}{\overset{R_3 - \frac{c}{a}R_1 \rightarrow R_1}{\Huge \sim}} \begin{bmatrix} a \amp 0 \amp b \amp 0 \amp \rvert \amp 1 \\ 0 \amp a \amp 0 \amp b \amp \rvert \amp 0 \\ 0 \amp 0 \amp d - \frac{bc}{a} \amp 0 \amp \rvert \amp -\frac{c}{a} \\ 0 \amp 0 \amp 0 \amp d - \frac{bc}{a} \amp \rvert \amp 1 \end{bmatrix} \end{equation*}
Solving for \(x_4\text{,}\) we see that
\begin{equation*} x_4 = \frac{1}{d-\frac{bc}{a}} = \frac{1}{\frac{ad - bc}{a}} = \frac{a}{ad - bc}\text{.} \end{equation*}
Solving for \(x_3\text{,}\) we see that
\begin{equation*} x_3 = \frac{-\frac{c}{a}}{d - \frac{bc}{a}} = \frac{-\frac{c}{a}}{\frac{ad-bc}{a}} = -\frac{c}{a}\left(\frac{a}{ad-bc}\right) = \frac{-c}{ad-bc} \end{equation*}
Solving for \(x_2\text{,}\) we see that
\begin{equation*} x_2 = \frac{-bx_4}{a} = \frac{-b\left(\frac{a}{ad-bc}\right)}{a} = \frac{-b}{ad-bc} \end{equation*}
Lastly, solving for \(x_1\text{,}\) we see that
\begin{equation*} x_1 = \frac{1 - bx_3}{a} = \frac{1 - b\left(\frac{-c}{ad-bc}\right)}{a} = \frac{\frac{ad-bc}{ad-bc} + \frac{bc}{ad-bc}}{a} = \frac{\frac{ad}{ad-bc}}{a} = \frac{d}{ad-bc}\text{.} \end{equation*}
It follows that
\begin{equation*} A^{-1} = \begin{bmatrix} x_1 \amp x_2 \\ x_3 \amp x_4 \end{bmatrix} = \begin{bmatrix} \frac{d}{ad-bc} \amp \frac{-b}{ad-bc} \\ \frac{-c}{ad-bc} \amp \frac{a}{ad-bc} \end{bmatrix} = \frac{1}{ad-bc}\begin{bmatrix} d \amp -b \\ -c \amp a \end{bmatrix}\text{.} \end{equation*}
