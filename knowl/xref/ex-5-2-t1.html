Task 5.23.a.
The only eigenvalue is \(\lambda=0\text{,}\) but the geometric multiplicity of \(\lambda=0\) is one.
🔗
Solution.
For simplicity, we will attempt to form an upper triangular matrix with the properties. This makes things easier since we know that the eigenvalues of a triangular matrix are exactly the entries on the diagonal. Therefore, each diagonal element of our upper triangular matrix needs to be 0. This leads us to
\begin{equation*} A = \begin{bmatrix} 0 \amp \amp \amp \amp \\ 0 \amp 0 \amp \amp \amp \\ 0 \amp 0 \amp 0 \amp \amp \\ 0 \amp 0 \amp 0 \amp 0 \amp \\ 0 \amp 0 \amp 0 \amp 0 \amp 0\end{bmatrix} \end{equation*}
where the blank entries are undecided. So far, we have guaranteed that every eigenvalue of \(A\) is \(\lambda = 0\text{.}\) Now, we need to force \(\lambda = 0\) to have geometric multiplicity equal to 1. This means that the matrix \(A - 0I = A\) needs to have exactly one free variable. Note that, as is, Column 1 of \(A\) already corresponds to a free variable. Thus, we need to make sure that Columns 2, 3, 4, and 5 are pivot columns. To do that, we need to put nonzero entries directly above the diagonal entry. This give the matrix
\begin{equation*} A = \begin{bmatrix} 0 \amp 1 \amp \amp \amp \\ 0 \amp 0 \amp 2 \amp \amp \\ 0 \amp 0 \amp 0 \amp 3 \amp \\ 0 \amp 0 \amp 0 \amp 0 \amp 4 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0\end{bmatrix}\text{.} \end{equation*}
We can fill in the remaining entries with whatever we want and \(A\) will still have the required properties.
