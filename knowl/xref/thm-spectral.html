Theorem 7.18. Spectral Theorem.

If \(A\) is an \(n\times n\) symmetric matrix, then:
🔗
  1. All eigenvalues of \(A\) are real.
    🔗
  2. The dimension of each eigenspace equals its algebraic multiplicity.
    🔗
  3. Eigenspaces corresponding to distinct eigenvalues are orthogonal.
    🔗
  4. \(A\) can be orthogonally diagonalized as \(A=PDP^T\text{,}\) where \(P\) is orthogonal.
    🔗
Solution.
Note that only symmetric matrices can be orthogonally diagonalized. This is because if \(A\) is a matrix that can be diagonalized as \(A = PDP^T\text{,}\) then
\begin{equation*} A^T = (PDP^T)^T = (P^T)^TD^TP^T = PDP^T = A\text{,} \end{equation*}
so \(A\) is symmetric.
