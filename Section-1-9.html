    Definition 1.56. The Standard Vectors.

    The standard vectors in \(\mathbb{R}^n\) are the vectors \(\vec e_1, \vec e_2, \ldots, \vec e_n\text{.}\)
    ğŸ”—
    They are:
    ğŸ”—
    \begin{equation*} \vec e_1 = \begin{bmatrix} 1 \\ 0 \\ \vdots \\ 0 \end{bmatrix}, \qquad \vec e_2 = \begin{bmatrix} 0 \\ 1 \\ \vdots \\ 0 \end{bmatrix}, \qquad \ldots, \qquad \vec e_n = \begin{bmatrix} 0 \\ 0 \\ \vdots \\ 1 \end{bmatrix}. \end{equation*}
    For example, in \(\mathbb{R}^3\text{:}\)
    ğŸ”—
    \begin{equation*} \vec e_1=\begin{bmatrix}1\\0\\0\end{bmatrix}, \qquad \vec e_2=\begin{bmatrix}0\\1\\0\end{bmatrix}, \qquad \vec e_3=\begin{bmatrix}0\\0\\1\end{bmatrix}. \end{equation*}
    ğŸ”—
    Â 
    ğŸ”—
    Note: If \(A\) is an \(m\times n\) matrix with columns \(\vec v_1,\vec v_2,\ldots,\vec v_n\text{,}\) then
    ğŸ”—
    \begin{equation*} A\vec e_i = \vec v_i, \text{ for } i=1,2,\ldots,n. \end{equation*}
    So multiplying a matrix by \(\vec e_i\) gives column \(i\) of \(A\text{.}\)

    Theorem 1.58. The Standard Matrix of a Linear Transformation.

    Let \(T:\mathbb{R}^n \to \mathbb{R}^m\) be a linear transformation. Then there is a unique matrix \(A\) such that
    ğŸ”—
    \begin{equation*} T(\vec x)=A\vec x, \qquad \vec x\in \mathbb{R}^n. \end{equation*}
    In fact, \(A\) is an \(m\times n\) matrix, and its \(j\)th column is \(T(\vec e_j)\text{.}\) That is,
    ğŸ”—
    \begin{equation*} A= \begin{bmatrix} T(\vec e_1) \amp T(\vec e_2) \amp \cdots \amp T(\vec e_n) \end{bmatrix}. \end{equation*}
    ğŸ”—
    The matrix \(A\) is the standard matrix for the linear transformation \(T\text{.}\)

    ğŸ”—
    Solution.
    Let us determine \(A\) such that \(T(\vec x) = A\vec x \) is rotation by \(\theta\) degrees counterclockwise. By TheoremÂ 1.58, we need to determine \(T\left(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\right)\) and \(T\left(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\right)\text{.}\)
    ğŸ”—
    First, note that \(T\left(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\right)\) is the vector in \(\mathbb{R}^2\) obtained by rotating the vector \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\text{,}\) which points in the direction of \(0^\circ\text{,}\) by \(\theta\) degrees counterclockwise. The resulting vector would have terminal end at the point on the unit circle whose angle is \(\theta\text{.}\) Using trigonometry, this point is \((\cos(\theta), \sin(\theta))\text{.}\) Therefore, we have that
    \begin{equation*} T\left(\begin{bmatrix}1 \\ 0 \end{bmatrix}\right) = \begin{bmatrix} \cos(\theta) \\ \sin(\theta) \end{bmatrix}\text{.} \end{equation*}
    ğŸ”—
    Next, note that \(T\left(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\right)\) is the vector in \(\mathbb{R}^2\) obtained by rotating the vector \(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\text{,}\) which points in the direction of \(90^\circ\text{,}\) by \(\theta\) degrees counterclockwise. The resulting vector would have terminal end at the point on the unit circle whose angle is \(90^\circ + \theta\text{.}\) It follows that this terminal point is \((\cos(90^\circ + \theta), \sin(90^\circ + \theta))\text{.}\) Using the sum rule for sine and cosine, we see that this terminal point can be expressed as \((-\sin(\theta), \cos(\theta))\text{.}\) Therefore, we have that
    \begin{equation*} T\left(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\right) = \begin{bmatrix} -\sin(\theta) \\ \cos(\theta) \end{bmatrix}\text{.} \end{equation*}
    ğŸ”—
    Thus, we have that the matrix \(A \) so that \(T(\vec x) = A\vec x \) results in rotating the vector \(\vec x\) by \(\theta\) degrees counterclockwise is
    \begin{equation*} A = \begin{bmatrix} \cos(\theta) \amp -\sin(\theta) \\ \sin(\theta) \amp \cos(\theta) \end{bmatrix}\text{.} \end{equation*}
    ğŸ”—
    Letâ€™s choose "nice" values of \(\theta\) to test out our matrix \(A\text{.}\)
    • If \(\theta = 0^\circ\text{,}\) then
      \begin{equation*} A = \begin{bmatrix} \cos(0^\circ) \amp -\sin(0^\circ) \\ \sin(0^\circ) \amp \cos(0^\circ) \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix}\text{.} \end{equation*}
      Note that \(T(\vec x) = A\vec x = \vec x\) for all \(\vec x \in \mathbb{R}^2\text{,}\) so \(T\) does not change any input vector \(\vec x\text{.}\) This makes sense since rotating a vector by \(0^\circ\) doesnâ€™t change the vector.
      ğŸ”—
      ğŸ”—

      \begin{equation*} A = \begin{bmatrix} \cos(180^\circ) \amp -\sin(180^\circ) \\ \sin(180^\circ) \amp \cos(180^\circ) \end{bmatrix} = \begin{bmatrix} -1 \amp 0 \\ 0 \amp -1 \end{bmatrix}\text{.} \end{equation*}
      Note that \(T(\vec x) = A\vec x = -\vec x\) for all \(\vec x \in \mathbb{R}^2\text{,}\) so \(T\) makes each vector \(\vec x\) point in the opposite direction. This makes sense since rotating a vector by \(180^\circ\) also makes vectors point in the opposite direction.
      ğŸ”—

        which points in the direction of \(180^\circ\)
        ğŸ”—

    Definition 1.60. Onto.

    A linear transformation \(T:\mathbb{R}^n \to \mathbb{R}^m\) is onto if for every \(\vec b\in \mathbb{R}^m\) there exists \(\vec x\in \mathbb{R}^n\) such that \(T(\vec x)=\vec b\text{.}\) In otherwords, a linear transformation is onto if the range of \(T\) equals the codomain of \(T\text{.}\)
    ğŸ”—
    ğŸ”—
    Onto is an existence property: for any \(\vec b\in\mathbb{R}^m\text{,}\) the system \(A\vec x=\vec b\) has a solution (where \(A\) is the standard matrix of \(T\)).
    ğŸ”—
    Examples
    ğŸ”—
    • A rotation on the plane is onto.
      ğŸ”—
    • A projection in the plane is not onto.
      ğŸ”—

    Theorem 1.61. Onto: Equivalent Statements.

    For a linear transformation \(T:\mathbb{R}^n \to \mathbb{R}^m\) with standard matrix \(A\text{,}\) the following are equivalent:
    ğŸ”—
    1. \(T\) is onto.
      ğŸ”—
    2. The columns of \(A\) span \(\mathbb{R}^m\text{.}\)
      ğŸ”—
    3. The matrix \(A\) has a pivot in every row (equivalently, \(m\) pivotal columns).

    Definition 1.62. One-to-One.

    A linear transformation \(T:\mathbb{R}^n \to \mathbb{R}^m\) is one-to-one if for every \(\vec b\in\mathbb{R}^m\) there is at most one (possibly none) \(\vec x\in\mathbb{R}^n\) such that \(T(\vec x)=\vec b\text{.}\)
    ğŸ”—
    ğŸ”—
    One-to-one is a uniqueness property; it does not assert existence for all \(\vec b\text{.}\)
    ğŸ”—
    Examples
    ğŸ”—
    • A rotation on the plane is one-to-one.
      ğŸ”—
    • A projection in the plane is not one-to-one.
      ğŸ”—

    Theorem 1.63. One-to-One: Equivalent Statements.

    For a linear transformation \(T:\mathbb{R}^n \to \mathbb{R}^m\) with standard matrix \(A\text{,}\) the following are equivalent:
    ğŸ”—
    1. \(T\) is one-to-one.
      ğŸ”—
    2. The unique solution to \(T(\vec x)=\vec 0\) is \(\vec x=\vec 0\text{.}\)
      ğŸ”—
    3. The columns of \(A\) are linearly independent.
      ğŸ”—
    4. Every column of \(A\) is pivotal (equivalently, \(A\) has no free variables).

    ğŸ”—
    (a)
    \(A\) is a \(2\times 3\) standard matrix for a one-to-one linear transformation.
    \begin{equation*} A= \begin{bmatrix} 1 \amp 0 \amp \\ 0 \amp \amp 1 \end{bmatrix}. \end{equation*}
    Solution.
    It is not possible to make \(A\) the standard matrix for a one-to-one linear transformation since doing so would require \(A\) to have a pivot in every column which cannot happen since the number of pivots is at most 2 (restricted by the number of rows). In fact, it is never possible for a \(2 \times 3\) matrix to represent a one-to-one linear transformation. Geometrically, this means that no linear transformation from \(\mathbb{R}^3\) to \(\mathbb{R}^2\) can bo one-to-one. This should make sense to be impossible since there are more input vectors (3 dimensions worth) than output vectors (2 dimensions worth) so many input vectors will correspond to the same output vector.
    ğŸ”—
    In this example, no matter how you fill in the blanks, you will always have a pivot in each row of \(A\text{.}\) Therefore, \(A\) will definitely represent an onto linear transformation.
    ğŸ”—
    ğŸ”—

    (b)
    \(B\) is a \(3\times 2\) standard matrix for an onto linear transformation.
    \begin{equation*} B= \begin{bmatrix} 1 \amp \\ \amp \\ \amp \end{bmatrix}. \end{equation*}

    Example 1.65.

    Define a linear transformation by
    ğŸ”—
    \begin{equation*} T(x_1,x_2)=(3x_1+x_2,\; 5x_1+7x_2,\; x_1+3x_2). \end{equation*}
    Is \(T\) one-to-one? Is \(T\) onto?
    ğŸ”—
    Solution.
    Note that
    \begin{equation*} T\left(\begin{bmatrix}x_1 \\ x_2 \end{bmatrix}\right) = \begin{bmatrix} 3x_1 + x_2 \\ 5x_1 + 7x_2 \\ x_1 + 3x_2 \end{bmatrix} = \begin{bmatrix} 3 \amp 1 \\ 5 \amp 7 \\ 1 \amp 3 \end{bmatrix}\begin{bmatrix} x_1 \\ x_2 \end{bmatrix}\text{.} \end{equation*}
    Thus, the matrix that represent \(T\) is
    \begin{equation*} A = \begin{bmatrix} 3 \amp 1 \\ 5 \amp 7 \\ 1 \amp 3 \end{bmatrix}\text{.} \end{equation*}
    Note that
    \begin{equation*} \begin{bmatrix} 3 \amp 1 \\ 5 \amp 7 \\ 1 \amp 3 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 3 \\ 5 \amp 7 \\ 3 \amp 1 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 3 \\ 0 \amp -8 \\ 0 \amp -8 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 3 \\ 0 \amp 1 \\ 0 \amp 0 \end{bmatrix} \end{equation*}
    has a pivot in each column, but not a pivot in each row. Therefore, \(T\) is a linear transformation which is one-to-one, but not onto.
