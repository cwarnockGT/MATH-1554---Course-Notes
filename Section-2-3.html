    Theorem 2.26. The Invertible Matrix Theorem (IMT).

    Let \(A\) be an \(n\times n\) matrix. The following statements are equivalent.
    ðŸ”—
    1. \(A\) is invertible.
      ðŸ”—
    2. \(A\) is row equivalent to \(I_n\text{.}\)
      ðŸ”—
    3. \(A\) has \(n\) pivotal columns.
      ðŸ”—
    4. \(A\vec x=\vec 0\) has only the trivial solution.
      ðŸ”—
    5. The columns of \(A\) are linearly independent.
      ðŸ”—
    6. The linear transformation \(\vec x \mapsto A\vec x\) is one-to-one.
      ðŸ”—
    7. The equation \(A\vec x=\vec b\) has a solution for all \(\vec b\in\mathbb{R}^n\text{.}\)
      ðŸ”—
    8. The columns of \(A\) span \(\mathbb{R}^n\text{.}\)
      ðŸ”—
    9. The linear transformation \(\vec x \mapsto A\vec x\) is onto.
      ðŸ”—
    10. There is an \(n\times n\) matrix \(C\) such that \(CA=I_n\text{.}\) (\(A\) has a left inverse.)
      ðŸ”—
    11. There is an \(n\times n\) matrix \(D\) such that \(AD=I_n\text{.}\) (\(A\) has a right inverse.)
      ðŸ”—
    12. \(A^T\) is invertible.

    Theorem 2.27. Left and Right Inverses.

    If \(A\) and \(B\) are \(n\times n\) matrices and \(AB=I\text{,}\) then both matrices are invertible, with \(B=A^{-1}\) and \(A=B^{-1}\text{.}\)

    Example 2.28.

    Is \(A\) invertible?
    ðŸ”—
    \begin{equation*} A = \begin{bmatrix} 1 \amp 0 \amp -2\\ 3 \amp 1 \amp -2\\ -5 \amp -1 \amp 9 \end{bmatrix} \end{equation*}
    Solution.
    We can use IMT to determine if \(A\) is invertible by showing that \(A\) either makes at least one statement in the list true or makes at least one statement in the list false. Note that
    \begin{equation*} A = \begin{bmatrix} 1 \amp 0 \amp -2 \\ 3 \amp 1 \amp -2 \\ -5 \amp -1 \amp 9 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 0 \amp -2 \\ 0 \amp 1 \amp 4 \\ 0 \amp -1 \amp -1 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 0 \amp -2 \\ 0 \amp 1 \amp 4 \\ 0 \amp 0 \amp 3 \end{bmatrix}\text{.} \end{equation*}
    We see that \(A \) has a pivot in every column. Therefore, by IMT, \(A \) must make every statment on the list true. In particular, we learn that \(A \) is invertible.

    Example 2.29.

    For each matrix below, determine all values of \(h\) so that the matrix is singular (not invertible). If there are no values of \(h \text{,}\) explain why.
    ðŸ”—
    (a)
    \(A = \begin{bmatrix} 1 \amp 0 \amp 1\\ 1 \amp h \amp 1\\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    ðŸ”—
    Solution.
    Note that
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 1 \\ 1 \amp h \amp 1 \\ 0 \amp 0 \amp 1 \end{bmatrix} \sim \begin{bmatrix} 1\amp 0 \amp 1 \\ 0 \amp h \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\text{.} \end{equation*}
    Thus, \(A \)is missing a pivot if and only if \(h = 0\text{.}\) Therefore, \(h = 0\) is the only value of \(h \) which makes \(A \) singular.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    (b)
    \(B = \begin{bmatrix} 1 \amp h \amp 1\\ 0 \amp 1 \amp 1\\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    ðŸ”—
    Solution.
    Note that regardless of the value of \(h \text{,}\) \(B \) will have a pivot in each column. Therefore, no value of \(h \) will make \(B \) singular.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    (c)
    \(C = \begin{bmatrix} 1 \amp 0 \amp 0\\ 0 \amp 1 \amp 1\\ 0 \amp h \amp 1 \end{bmatrix}\)
    ðŸ”—
    Solution.
    Note that
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 1 \\ 0 \amp h \amp 1 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 1 \\ 0 \amp h-1 \amp 0\end{bmatrix}\text{.} \end{equation*}
    Thus, \(C \) is missing a pivot in the third row if and only if \(h = 1\text{.}\) Therefore, \(h = 1\) is the only value of \(h\) which makes \(C \) singular.
